// IDW Formula: Z = Σ(Zi/di^p) / Σ(1/di^p)
// ── p = 1 (Linear) ──
Point 1: Z=160 d=1000 w=1/1000 = 0.00100 wZ = 0.160
Point 2: Z=170 d= 400 w=1/400 = 0.00250 wZ = 0.425
Point 3: Z=180 d= 600 w=1/600 = 0.00167 wZ = 0.300
Point 4: Z= 90 d= 120 w=1/120 = 0.00833 wZ = 0.750
Σw = 0.01350 ΣwZ = 1.635
Z(p=1) = 1.635 / 0.01350 = 121.11
// ── p = 2 (Quadratic — GIS default) ──
Point 1: w=1/1000² = 0.0000010 wZ = 0.000160
Point 2: w=1/400² = 0.0000063 wZ = 0.001063
Point 3: w=1/600² = 0.0000028 wZ = 0.000500
Point 4: w=1/120² = 0.0000694 wZ = 0.006250
Σw = 0.0000795 ΣwZ = 0.007973
Z(p=2) = 0.007973 / 0.0000795 = 100.32
// p=2 gives more weight to nearest point (Z=90 at d=120)
// That's why p=2 result is closer to 90

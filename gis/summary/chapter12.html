1 Visual Variables (7 Variables of Bertin)

  • Size: Symbol dimensions (larger = more). Best for quantitative.
  • Shape: Different forms (circle, triangle). For nominal data.
  • Hue: Different colours. Best for qualitative/categorical.
  • Value (Lightness): Light-to-dark shading. For ordered data.
  • Saturation: Intensity of colour (vivid vs dull).
  • Orientation: Direction of hatching/symbols.
  • Texture/Pattern: Density and arrangement of markings.

2 Essential Map Elements

  • Title: What, where, when. Prominent position.
  • Legend: Symbol key for all features and colours.
  • Scale bar: Visual ground distance. Graphical preferred.
  • North arrow: Map orientation.
  • Frame/Neatline: Boundary around map body.
  • Grid/Graticule: Reference lines.
  • Source/Citation: Data source, projection, datum.
  • Inset map: Location context or enlarged detail.

3 Map Types

  • Choropleth: Areas shaded by value. Best for normalised ratio data (density).
  • Graduated Symbol: Proportional symbols sized by value. Good for absolute counts.
  • Dot Density: Each dot = fixed quantity. Shows distribution pattern.
  • Isopleth/Isoline: Lines connecting equal values (contours, isotherms).
  • Flow Map: Lines showing movement direction and magnitude.

4 Map Design Principles

  • Visual Hierarchy: Most important features draw the eye first.
  • Balance: Elements distributed evenly.
  • Simplicity: Remove clutter. Every element serves a purpose.
  • Contrast: Use colour/value differences to distinguish features.
  • Harmony: Consistent fonts, colour scheme, style.
  • Figure-ground: Main subject stands out from background.
Q Past Exam Questions
+What are the visual variables used in cartographic representation? (2074, 2076, 2079)

All 7: size, shape, hue, value, saturation, orientation, texture. Draw small examples. Mention which suit nominal, ordinal, quantitative.

+List and explain the essential elements of a good map. (2072, 2075, 2080, 2082)

Title, legend, scale bar, north arrow, frame, grid, source, inset. Draw a sample layout showing placement.

+Compare choropleth and graduated symbol maps. (2075 Bhadra, 2081)

Choropleth: shaded areas, ratio data. Graduated: sized symbols, absolute counts. Choropleth can mislead with area sizes (MAUP).

+What are the principles of effective map design? (2073, 2078)

Hierarchy, balance, simplicity, contrast, harmony, figure-ground. Brief example for each.

Key Formulas & Process

1. Sink Detection

A sink is a cell whose elevation is lower than ALL 8 neighbours.

Check: if Z(cell) < min(8 neighbours) → it’s a sink.

Sinks trap water and disconnect river networks — they must be filled before hydrological analysis.

2. Sink Filling

Formula: Filled value = min(8 neighbours) + 0.5

This ensures water can flow out of the former sink to its lowest neighbour.

3. Flow Direction (D8)

Each cell flows to the neighbour with the steepest downhill slope.

For cardinal directions: slope = ΔZ / cell_size

For diagonal directions: slope = ΔZ / (cell_size × √2)

D8 Codes: E=1, SE=2, S=4, SW=8, W=16, NW=32, N=64, NE=128

4. Flow Accumulation

Count how many cells flow into each cell (upstream count).

Start from ridge cells (FA=0) and trace downstream, adding 1 for each contributing cell.

The cell with the highest FA is the outlet of the entire grid.

5. River Network

Formula: River threshold (cells) = Threshold area / cell area

Cell area = resolution × resolution (e.g., 30m × 30m = 900 m²)

All cells with FA ≥ threshold are river cells.

6. Watershed Area

Formula: Watershed area = Number of contributing cells × cell area

The watershed includes ALL cells that drain to the outlet point.

For the grid outlet: watershed area = total cells × cell area.

Standard Process (Exam Steps)

  1. Find sinks: Check each interior cell — is it lower than ALL 8 neighbours?
  2. Fill sinks: Raise to min_neighbour + 0.5
  3. Flow Direction: For each cell, find the neighbour with steepest drop
  4. Flow Accumulation: Count upstream cells flowing into each cell
  5. River Network: Convert threshold area to cells; mark FA ≥ threshold as river
  6. Watershed: Count all cells draining to the outlet; multiply by cell area

Key Formulas & Process

1. Inverse Distance Weighting (IDW)

IDW estimates the value at an unknown point using values from nearby known points, giving more weight to closer points.

Formula:

Z = Σ(Zi / dip) / Σ(1 / dip)

Where:

• Zi = value at known point i

• di = distance from unknown point to known point i

• p = power parameter (typically 1 or 2)

Key insight: Higher p → nearest point dominates more. When p=2 (GIS default), the nearest point has much more influence than distant ones.

2. Weighted Suitability Analysis

Combines multiple raster layers using weighted overlay.

Formula:

Score = Σ(Weighti × Ratingi)

Where:

• Weighti = importance of factor i (must sum to 1.0)

• Ratingi = suitability rating of the cell for factor i (typically 1–10)

• Suitable area = count of suitable cells × cell area

Process Steps

  1. IDW: Calculate weight wi = 1/dip for each known point
  2. IDW: Multiply each weight by its value: wi × Zi
  3. IDW: Sum weighted values and divide by sum of weights: Z = Σ(wiZi) / Σwi
  4. Suitability: Reclassify each input layer to a common scale (1–10)
  5. Suitability: Multiply each reclassified layer by its weight
  6. Suitability: Sum weighted layers, apply threshold to find suitable cells

// Sum of weighted values

Z(p=1) = 1.635 / 0.01350 = 121.11

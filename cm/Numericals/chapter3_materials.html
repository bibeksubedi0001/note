EOQ Technique Summary

Economic Order Quantity (EOQ) Formulas

EOQ Formula

EOQ = √(2 × D × Co) / (Ci × P)

Where: D = Annual demand, Co = Ordering cost per order, Ci = Carrying cost rate (fraction), P = Unit price

Related Formulas

Number of Orders = D / EOQ

Cycle Time = EOQ / d (d = daily consumption = D / working days)

Reorder Level (ROL) = (Lead Time × d) + Safety Stock

Max Stock = EOQ + Safety Stock

Min Stock = Safety Stock

Average Stock = EOQ/2 + Safety Stock

GiT (Goods in Transit) = Lead Time × d

Total Inventory Cost = (D/EOQ)×Co + (EOQ/2)×Ci×P

ABC Classification Technique

ABC Analysis Procedure

Steps
  1. Calculate Annual Usage Value = Quantity × Rate per unit for each material
  2. Rank materials in descending order of usage value
  3. Calculate cumulative percentage of total value
  4. Classify:
    • A: Top 70–80% of value (usually ~10–20% of items) — High control
    • B: Next 15–20% of value (~20–30% of items) — Medium control
    • C: Remaining 5–10% of value (~50–70% of items) — Low control

Q2. EOQ & Inventory 2079 Jestha

Annual cement = 6000 bags, lead time = 5 days, carrying cost = 15%, working days = 200/year, ordering cost = Rs.25,000/order, safety stock = 190 bags. Find EOQ, GiT, ROL. [6 marks]

Solution

Given

D = 6,000 bags/year, LT = 5 days, Ci = 15% = 0.15, Working days = 200, Co = Rs.25,000

Safety Stock (SS) = 190 bags

Note: Unit price P is not given directly. For cement, we can assume a reasonable rate. However, EOQ formula without P uses: EOQ = √(2DCo/Ch) where Ch = carrying cost per unit per year = Ci × P.

Since P isn't given, we use the carrying cost per unit per year. Assuming cement rate ≈ Rs.750/bag (common rate):

Ch = 0.15 × 750 = Rs.112.5/bag/year

EOQ Calculation

EOQ = √(2 × 6000 × 25000 / 112.5)

EOQ = √(300,000,000 / 112.5) = √2,666,667

EOQ ≈ 1,633 bags

Daily Consumption

d = D / Working days = 6000 / 200 = 30 bags/day

Goods in Transit (GiT)

GiT = Lead Time × d = 5 × 30 = 150 bags

Yes, there are goods in transit = 150 bags

Reorder Level (ROL)

ROL = (LT × d) + SS = (5 × 30) + 190 = 150 + 190 = 340 bags

★ EOQ ≈ 1,633 bags
★ GiT = 150 bags (Yes, goods in transit exist)
★ Re-order Level = 340 bags

Q3. EOQ & Stock Levels 2078 Kartik

Calculate EOQ, Maximum Stock level, minimum, average stock level, ROP/ROL, and total inventory cost. Annual cement = 20,000 bags, Co = Rs.10,000, Ci = 5%, P = Rs.900/bag, safety stock = 10 days, lead time = 5 days. [6 marks]

Solution

Given

D = 20,000 bags/year, Co = Rs.10,000, Ci = 5% = 0.05, P = Rs.900/bag, LT = 5 days

Assuming 365 working days/year: d = 20,000/365 = 54.79 bags/day

Safety Stock = 10 days × 54.79 = 548 bags

EOQ

Ch = Ci × P = 0.05 × 900 = Rs.45/bag/year

EOQ = √(2 × 20,000 × 10,000 / 45)

EOQ = √(400,000,000 / 45) = √8,888,889

EOQ ≈ 2,981 bags

Stock Levels

Maximum Stock Level = EOQ + SS = 2,981 + 548 = 3,529 bags

Minimum Stock Level = SS = 548 bags

Average Stock Level = EOQ/2 + SS = 1,490.5 + 548 = 2,039 bags

Reorder Point (ROP/ROL)

ROL = (LT × d) + SS = (5 × 54.79) + 548 = 274 + 548 = 822 bags

Total Inventory Cost

Ordering Cost = (D/EOQ) × Co = (20,000/2,981) × 10,000 = 6.71 × 10,000 = Rs.67,100

Carrying Cost = (EOQ/2) × Ch = (2,981/2) × 45 = 1,490.5 × 45 = Rs.67,073

Total Inventory Cost = 67,100 + 67,073 = Rs.1,34,173

(Note: At EOQ, ordering cost ≈ carrying cost, which verifies our EOQ calculation.)

★ EOQ ≈ 2,981 bags
★ Max Stock = 3,529 bags | Min Stock = 548 bags | Avg Stock = 2,039 bags
★ ROL = 822 bags
★ Total Inventory Cost = Rs.1,34,173

Q4. EOQ & Reorder Point 2071 Bhadra

Calculate EOQ and reorder point. Rebar = 1000 tonnes, Co = Rs.10,000, Ci = 15%, P = Rs.90,000/tonne, SS = 50 tonnes, LT = 4 days, consumption = 50 tonnes/day. [Part of 6 marks]

Solution

Given

D = 1,000 tonnes, Co = Rs.10,000, Ci = 15% = 0.15, P = Rs.90,000/tonne

SS = 50 tonnes, LT = 4 days, d = 50 tonnes/day

EOQ

Ch = Ci × P = 0.15 × 90,000 = Rs.13,500/tonne/year

EOQ = √(2 × 1,000 × 10,000 / 13,500)

EOQ = √(20,000,000 / 13,500) = √1,481.48

EOQ ≈ 38.49 ≈ 39 tonnes

Reorder Point

ROL = (LT × d) + SS = (4 × 50) + 50 = 200 + 50 = 250 tonnes

★ EOQ ≈ 39 tonnes
★ Reorder Point = 250 tonnes

Q5. Simple EOQ 2070 Magh

Calculate EOQ of Cement. Total Quantity = 12,000 bags, Ci = 15%, Co = Rs.10,000, P = Rs.750/bag. [3 marks]

Solution

Given

D = 12,000 bags, Ci = 15% = 0.15, Co = Rs.10,000, P = Rs.750/bag

EOQ Calculation

Ch = Ci × P = 0.15 × 750 = Rs.112.5/bag/year

EOQ = √(2 × D × Co / Ch)

EOQ = √(2 × 12,000 × 10,000 / 112.5)

EOQ = √(240,000,000 / 112.5) = √2,133,333

EOQ ≈ 1,461 bags

★ EOQ ≈ 1,461 bags

Contract Management — Performance Guarantee

1 Solved Problem 2081

Performance Guarantee per PPR 2064

Rules (Public Procurement Regulation 2064)

Performance Guarantee Rate

As per PPR 2064, the contractor must submit a performance guarantee (also called performance security) for the following amounts:

  • If Bid Amount ≤ Estimated Cost: Performance Guarantee = 5% of Bid Amount
  • If Bid Amount > Estimated Cost: Performance Guarantee = 5% of Bid Amount + 50% of (Bid Amount − Estimated Cost)

The extra 50% provision ensures protection when the bid is significantly above the estimated cost.

Q1. Performance Guarantee 2081 Shrawan

Calculate performance guarantee amount require for a contractor whose bidding amount is Rs 1,00,50,000. The estimated cost of the project is Rs 2,00,00,000. As per PPR 2064. [2 marks]

Solution

Given

Bid Amount (BA) = Rs.1,00,50,000

Estimated Cost (EC) = Rs.2,00,00,000

Check Condition

BA = Rs.1,00,50,000 < EC = Rs.2,00,00,000

Since Bid Amount < Estimated Cost, we use the simple formula.

Calculation

$$\text{Performance Guarantee} = 5\% \text{ of Bid Amount}$$

$$\text{PG} = 0.05 \times 1{,}00{,}50{,}000$$

$$\boxed{\text{PG} = \text{Rs.}5{,}02{,}500}$$

★ Performance Guarantee = Rs.5,02,500
Alternative Scenario (For Practice)

If the bid amount were higher than estimated cost, say BA = Rs.2,50,00,000:

$$\text{PG} = 5\% \text{ of BA} + 50\% \text{ of } (\text{BA} - \text{EC})$$

$$= 0.05 \times 2{,}50{,}00{,}000 + 0.50 \times (2{,}50{,}00{,}000 - 2{,}00{,}00{,}000)$$

$$= 12{,}50{,}000 + 25{,}00{,}000 = \textbf{Rs.}37{,}50{,}000$$

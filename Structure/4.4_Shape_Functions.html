Collected & Prepared By: Er. Sanjay Kumar Sah (Lecturer, Department of Civil Engineering, Pulchowk Campus)

Shape Function / Interpolation Function

  • In finite element analysis, field variables at nodal points are found by rigorous analysis.
  • But field variables (especially basic variables) at any point inside the element is the function of values at nodal points of the element.
  • This function which relates the field variable at any point within the element to the field variables of nodal points, is called a shape function. The field variables approximated by the shape function are not as exact as obtained by rigorous analysis at the same point, and hence, it is also called an “Interpolation function.”

→ In two dimensional stress analysis, where the basic variable is displacement, it is given by:

$$u = \sum N_i u_i$$ $$v = \sum N_i v_i$$

where summation is over the number of nodes.

For example, for a three-noded triangular element, displacement at $P(x,y)$ is given by:

$$u = \sum_{i=1}^{3} N_i u_i = N_1 u_1 + N_2 u_2 + N_3 u_3$$ $$v = \sum_{i=1}^{3} N_i v_i = N_1 v_1 + N_2 v_2 + N_3 v_3$$

In matrix form:

$$\begin{Bmatrix} u \\ v \end{Bmatrix} = \begin{bmatrix} N_1 & N_2 & N_3 & 0 & 0 & 0 \\ 0 & 0 & 0 & N_1 & N_2 & N_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \\ v_1 \\ v_2 \\ v_3 \end{Bmatrix}$$ $$\{u\}_{2 \times 1} = [N]_{2 \times 6} \{u_i\}_{6 \times 1}$$
Definitions
  • $\{u\}$ — displacement vector at any point in the element.
  • $[N]$ — shape function.
  • $\{u_i\}$ — nodal displacement vector.

Polynomial Shape Function

Polynomial functions are commonly used as shape functions because of the following benefits:

  • They are easy to handle mathematically, i.e., differentiation and integration of polynomials is easy.
  • Any function can be approximated well by using a polynomial function. If a function is highly nonlinear, we may have to approximate with a higher-order polynomial.

Approximation with polynomial functions of different order:

One Dimensional Polynomial Shape Function

A general one-dimensional polynomial shape function of $n^{th}$ order (with $n+1$ terms) is given by:

$$u(x) = \alpha_1 + \alpha_2 x + \alpha_3 x^2 + \dots + \alpha_{n+1} x^n$$

In matrix form:

$$u = [G] \{\alpha\}$$

where $[G] = \begin{bmatrix} 1 & x & x^2 & \dots & x^n \end{bmatrix}$ and $\{\alpha\}^T = \begin{bmatrix} \alpha_1 & \alpha_2 & \alpha_3 & \dots & \alpha_{n+1} \end{bmatrix}$

Two Dimensional Polynomial Shape Function

A general form of two-dimensional polynomial shape function of $n^{th}$ order is given by:

$$u(x,y) = \alpha_1 + \alpha_2 x + \alpha_3 y + \alpha_4 x^2 + \alpha_5 xy + \alpha_6 y^2 + \alpha_7 x^3 + \dots + \alpha_m y^n$$ $$v(x,y) = \alpha_{m+1} + \alpha_{m+2} x + \alpha_{m+3} y + \alpha_{m+4} x^2 + \alpha_{m+5} xy + \dots + \alpha_{2m} y^n$$

In matrix form:

$$\{\delta\} = \begin{Bmatrix} u(x,y) \\ v(x,y) \end{Bmatrix} = [G_1]\{\alpha\} = \begin{bmatrix} G & 0 \\ 0 & G \end{bmatrix} \{\alpha\}$$

where $G = \begin{bmatrix} 1 & x & y & x^2 & xy & y^2 & x^3 & \dots & y^n \end{bmatrix}$ and $\{\alpha\}^T = \begin{bmatrix} \alpha_1 & \alpha_2 & \dots & \alpha_m & \alpha_{m+1} & \dots & \alpha_{2m} \end{bmatrix}$

The total number of terms ‘$m$’ in an $n^{th}$ degree polynomial is:

$$m = \frac{(n+1)(n+2)}{2}$$
  • For 1st order polynomial ($n = 1$): $m = \frac{(1+1)(1+2)}{2} = 3$ — Terms: $\alpha_1 + \alpha_2 x + \alpha_3 y$
  • For 2nd order polynomial ($n = 2$): $m = \frac{(2+1)(2+2)}{2} = 6$ — Terms: $\alpha_1 + \alpha_2 x + \alpha_3 y + \alpha_4 x^2 + \alpha_5 xy + \alpha_6 y^2$

Convergence Requirements of Shape Functions

→ The displacements obtained by finite element analysis are approximate solutions.
→ As the number of finite elements (mesh or no. of nodes) increases, the solution approaches the exact solutions.

In order to ensure convergence criteria, the shape functions should satisfy the following requirements:

i) Compatibility Requirement:

  • The displacement models must be continuous within the elements and displacements must be compatible between the adjacent elements.
  • The adjacent elements must deform without causing opening, overlaps or discontinuities between the elements.

ii) Rigid Body Displacements:

  • In a displacement model there should be a term which permits all points on the element to experience the same displacement.
  • If such a term does not exist, shifting of the origin of the coordinate system will cause additional stresses and strains, which should not occur.
  • In the displacement model $u = \alpha_1 + \alpha_2 x + \alpha_3 y$, the term $\alpha_1$ provides for the rigid body displacement.
  • Hence, to satisfy the requirement of rigid body displacement, there should be a constant term in the shape function selected.

iii) Constant Strain State:

  • The displacement model should be such that all points in the element can experience the same strain.
  • If we imagine the refinement of mesh, i.e., if elements approach infinitesimal size, the strains within the element approach constant values.
  • Unless the shape function term includes these constant strain terms, we cannot hope to converge to a correct solution.
  • In displacement model $u = \alpha_1 + \alpha_2 x + \alpha_3 y + \dots$, $\alpha_2$ and $\alpha_3$ provide for uniform strain $\epsilon_x$, and $\alpha_{m+3}$ provides for uniform strain $\epsilon_y$.

iv) Geometric Isotropy / Spatial Isotropy / Geometric Invariance:

  • The selection of polynomial shape function for the displacement model should be such that the pattern should be independent of the orientation of the local coordinate system.

Shape Functions Using Polynomials (Cartesian Coordinates)

Example 1: Shape function of a two-noded bar/truss element
Solution

Using generalized coordinate approach (Cartesian Coordinate).

Nodal variables are $u_1$ & $u_2$ along x-axis. Select a polynomial with two terms:

$$u = \alpha_1 + \alpha_2 x = \begin{bmatrix} 1 & x \end{bmatrix} \begin{Bmatrix} \alpha_1 \\ \alpha_2 \end{Bmatrix}$$

Applying boundary conditions:

$$u_1 = \alpha_1 + \alpha_2 x_1, \qquad u_2 = \alpha_1 + \alpha_2 x_2$$
$$\begin{Bmatrix} u_1 \\ u_2 \end{Bmatrix} = \begin{bmatrix} 1 & x_1 \\ 1 & x_2 \end{bmatrix} \begin{Bmatrix} \alpha_1 \\ \alpha_2 \end{Bmatrix}$$
$$\begin{Bmatrix} \alpha_1 \\ \alpha_2 \end{Bmatrix} = \frac{1}{l} \begin{bmatrix} x_2 & -x_1 \\ -1 & 1 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \end{Bmatrix} \quad --- (ii)$$

Substituting (ii):

$$u = \begin{bmatrix} 1 & x \end{bmatrix} \times \frac{1}{l} \begin{bmatrix} x_2 & -x_1 \\ -1 & 1 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \end{Bmatrix} = \frac{1}{l} \begin{bmatrix} x_2 - x & x - x_1 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \end{Bmatrix}$$
$$u = \begin{bmatrix} N_1 & N_2 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \end{Bmatrix} = N_1 u_1 + N_2 u_2$$
$$N_1 = \frac{x_2 - x}{l}, \qquad N_2 = \frac{x - x_1}{l}$$ $$[N] = \begin{bmatrix} N_1 & N_2 \end{bmatrix} = \begin{bmatrix} \frac{x_2 - x}{l} & \frac{x - x_1}{l} \end{bmatrix}$$

The values of $N_1$ and $N_2$ vary linearly.

Example 2: Shape functions for the Constant Strain Triangle (CST) using polynomial functions
Solution
CST with 6 nodal displacements

There are 6 nodal displacements $u_1, u_2, u_3, v_1, v_2, v_3$. Select polynomials with total 6 constants:

$$u = \alpha_1 + \alpha_2 x + \alpha_3 y$$ $$v = \alpha_4 + \alpha_5 x + \alpha_6 y$$

Applying boundary conditions for $u$:

$$\begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix} = \begin{bmatrix} 1 & x_1 & y_1 \\ 1 & x_2 & y_2 \\ 1 & x_3 & y_3 \end{bmatrix} \begin{Bmatrix} \alpha_1 \\ \alpha_2 \\ \alpha_3 \end{Bmatrix}$$
$$\begin{Bmatrix} \alpha_1 \\ \alpha_2 \\ \alpha_3 \end{Bmatrix} = \frac{1}{2A} \begin{bmatrix} a_1 & a_2 & a_3 \\ b_1 & b_2 & b_3 \\ c_1 & c_2 & c_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix}$$

where $\det \begin{vmatrix} 1 & x_1 & y_1 \\ 1 & x_2 & y_2 \\ 1 & x_3 & y_3 \end{vmatrix} = 2A$ and $A$ = area of triangle.

Constants

$a_1 = x_2 y_3 - x_3 y_2$;   $b_1 = y_2 - y_3$;   $c_1 = x_3 - x_2$

$a_2 = x_3 y_1 - x_1 y_3$;   $b_2 = y_3 - y_1$;   $c_2 = x_1 - x_3$

$a_3 = x_1 y_2 - x_2 y_1$;   $b_3 = y_1 - y_2$;   $c_3 = x_2 - x_1$

Substituting back:

$$u = \begin{bmatrix} 1 & x & y \end{bmatrix} \times \frac{1}{2A} \begin{bmatrix} a_1 & a_2 & a_3 \\ b_1 & b_2 & b_3 \\ c_1 & c_2 & c_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix}$$
$$u = \begin{bmatrix} N_1 & N_2 & N_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix} = N_1 u_1 + N_2 u_2 + N_3 u_3$$

where $[N] = \begin{bmatrix} N_1 & N_2 & N_3 \end{bmatrix}$ is the shape function.

Similarly, $v = \begin{bmatrix} N_1 & N_2 & N_3 \end{bmatrix} \begin{Bmatrix} v_1 \\ v_2 \\ v_3 \end{Bmatrix}$

$$\{\delta\} = \begin{Bmatrix} u \\ v \end{Bmatrix} = \begin{bmatrix} N_1 & N_2 & N_3 & 0 & 0 & 0 \\ 0 & 0 & 0 & N_1 & N_2 & N_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \\ v_1 \\ v_2 \\ v_3 \end{Bmatrix}$$

Shape Function using Lagrange Polynomials

If only continuity of basic unknown (displacement) is to be satisfied, Lagrange polynomials can be used to derive shape functions. Lagrange polynomial of one dimension is defined by:

$$L_k(x) = \prod_{\substack{m=1 \\ m \neq k}}^{n} \frac{(x - x_m)}{(x_k - x_m)}$$

Thus, if $n = 5$ and $k = 3$:

$$L_3(x) = \frac{(x - x_1)(x - x_2)(x - x_4)(x - x_5)}{(x_3 - x_1)(x_3 - x_2)(x_3 - x_4)(x_3 - x_5)}$$
Example 1: Shape function of two-noded bar element using Lagrange polynomials
Solution
$$N_1 = L_1 = \frac{x - x_2}{x_1 - x_2} = \frac{x_2 - x}{x_2 - x_1}$$ $$N_2 = L_2 = \frac{x - x_1}{x_2 - x_1}$$
Example 2: Shape function of three-noded bar element using Lagrange polynomials
Solution
$$N_1 = L_1 = \frac{(x - x_2)(x - x_3)}{(x_1 - x_2)(x_1 - x_3)}$$ $$N_2 = L_2 = \frac{(x - x_1)(x - x_3)}{(x_2 - x_1)(x_2 - x_3)}$$ $$N_3 = L_3 = \frac{(x - x_1)(x - x_2)}{(x_3 - x_1)(x_3 - x_2)}$$
Example 3: Shape function of 4-noded bar element using Lagrange polynomials
Solution
$$N_1 = \frac{(x - x_2)(x - x_3)(x - x_4)}{(x_1 - x_2)(x_1 - x_3)(x_1 - x_4)}$$ $$N_2 = \frac{(x - x_1)(x - x_3)(x - x_4)}{(x_2 - x_1)(x_2 - x_3)(x_2 - x_4)}$$ $$N_3 = \frac{(x - x_1)(x - x_2)(x - x_4)}{(x_3 - x_1)(x_3 - x_2)(x_3 - x_4)}$$ $$N_4 = \frac{(x - x_1)(x - x_2)(x - x_3)}{(x_4 - x_1)(x_4 - x_2)(x_4 - x_3)}$$

Lagrange Polynomial Approach for 2D/3D Elements

We may extend the concept to two and three dimensions by forming the products of the functions which hold good for the individual one-dimensional coordinate directions. For 2D element:

$$N_i = L_i(\xi) \cdot L_i(\eta)$$
Example 1: Shape functions for 4-noded rectangular element using Lagrange polynomial

(Only two nodes in each direction)

Solution
$$N_1 = L_1(\xi) L_1(\eta) = \frac{(\xi - 1)}{(-1 - 1)} \times \frac{(\eta - 1)}{(-1 - 1)} = \frac{1}{4}(1 - \xi)(1 - \eta)$$
$$N_2 = L_2(\xi) L_2(\eta) = \frac{(\xi + 1)}{(1 + 1)} \times \frac{(\eta - 1)}{(-1 - 1)} = \frac{1}{4}(1 + \xi)(1 - \eta)$$

Similarly:

$$N_1 = \frac{1}{4}(1 - \xi)(1 - \eta), \qquad N_2 = \frac{1}{4}(1 + \xi)(1 - \eta)$$ $$N_3 = \frac{1}{4}(1 + \xi)(1 + \eta), \qquad N_4 = \frac{1}{4}(1 - \xi)(1 + \eta)$$ $$\boxed{N_i = \frac{1}{4}(1 + \xi\xi_i)(1 + \eta\eta_i)}$$
Example 2: Shape functions for 9-noded rectangular element using Lagrange polynomial
Solution

$N_i = L_i(\xi) L_i(\eta)$ where there are 3 nodes in each direction ($n=3$).

Corner Nodes ($\xi = \pm 1$ & $\eta = \pm 1$):

$$N_1 = \frac{\xi(\xi - 1)\eta(\eta - 1)}{4}, \quad N_2 = \frac{\xi(\xi + 1)\eta(\eta - 1)}{4}$$ $$N_3 = \frac{\xi(\xi + 1)\eta(\eta + 1)}{4}, \quad N_4 = \frac{\xi(\xi - 1)\eta(\eta + 1)}{4}$$

Nodes 5 & 7 ($\xi = 0$, $\eta = \pm 1$):

$$N_5 = \frac{(\xi^2 - 1)\eta(\eta - 1)}{-2}, \qquad N_7 = \frac{(\xi^2 - 1)\eta(\eta + 1)}{-2}$$

Nodes 6 & 8 ($\xi = \pm 1$, $\eta = 0$):

$$N_6 = \frac{\xi(\xi + 1)(\eta^2 - 1)}{-2}, \qquad N_8 = \frac{\xi(\xi - 1)(\eta^2 - 1)}{-2}$$

Central Node 9 ($\xi = 0$, $\eta = 0$):

$$N_9 = (\xi^2 - 1)(\eta^2 - 1)$$

General formulas:

Corner Nodes: $N_i = \frac{\xi(\xi+\xi_i)}{(-2)} \times \frac{\eta(\eta+\eta_i)}{(-2)}$

Nodes 5 & 7 ($\xi \neq \pm 1$, $\eta = \pm 1$): $N_i = (\xi^2-1) \times \frac{\eta(\eta+\eta_i)}{(-2)}$

Nodes 6 & 8 ($\xi = \pm 1$, $\eta \neq \pm 1$): $N_i = \frac{\xi(\xi+\xi_i)}{(-2)} \times (\eta^2-1)$

Central Node: $N_9 = (\xi^2-1)(\eta^2-1)$

Shape Function for Serendipity Family Elements

The shape functions are found from consideration that $N_i = 1$ for $i^{th}$ node and zero when referred to any other node.

Example 1: Shape function of 4-noded rectangular element using ‘serendipity concept’
Solution

Shape function $N_1$ has to satisfy:

  • $N_1 = 0$ @ $\xi = 1$ — (a)
  • $N_1 = 0$ @ $\eta = 1$ — (b)
  • $N_1 = 1$ @ $\xi = -1$ & $\eta = -1$ — (c)

Hence, assuming $N_1 = C(1 - \xi)(1 - \eta)$. Applying (c):

$$1 = C(1 + 1)(1 + 1) \Rightarrow C = \frac{1}{4}$$
$$N_1 = \frac{(1 - \xi)(1 - \eta)}{4}, \qquad N_2 = \frac{(1 + \xi)(1 - \eta)}{4}$$ $$N_3 = \frac{(1 + \xi)(1 + \eta)}{4}, \qquad N_4 = \frac{(1 - \xi)(1 + \eta)}{4}$$ $$\boxed{N_i = \frac{1}{4}(1 + \xi\xi_i)(1 + \eta\eta_i)}$$
Example 2: Shape function of quadratic serendipity rectangular element (8-noded)
Solution

Corner node $N_1$:

  • $N_1 = 0$ @ $\xi = 1$
  • $N_1 = 0$ @ $\eta = 1$
  • $N_1 = 0$ @ $\xi + \eta + 1 = 0$ (diagonal through 8 and 5)
  • $N_1 = 1$ @ $\xi = -1$ & $\eta = -1$

Assume: $N_1 = C(1 - \xi)(1 - \eta)(1 + \xi + \eta)$

$$1 = C(1+1)(1+1)(1-1-1) \Rightarrow C = -\frac{1}{4}$$
$$N_1 = -\frac{(1-\xi)(1-\eta)(1+\xi+\eta)}{4}$$

Similarly:

$$N_2 = -\frac{(1+\xi)(1-\eta)(1-\xi+\eta)}{4}$$ $$N_3 = -\frac{(1+\xi)(1+\eta)(1-\xi-\eta)}{4}$$ $$N_4 = -\frac{(1-\xi)(1+\eta)(1+\xi-\eta)}{4}$$

Mid-side node $N_5$:

  • $N_5 = 0$ @ $\xi = \pm 1$ and @ $\eta = 1$
  • $N_5 = 1$ @ $\xi = 0$ & $\eta = -1$

Assume: $N_5 = C(1-\xi)(1+\xi)(1-\eta)$

$$1 = C(1)(1)(2) \Rightarrow C = \frac{1}{2}$$
$$N_5 = \frac{(1-\xi^2)(1-\eta)}{2}$$

Similarly:

$$N_6 = \frac{(1+\xi)(1-\eta^2)}{2}, \quad N_7 = \frac{(1-\xi^2)(1+\eta)}{2}, \quad N_8 = \frac{(1-\xi)(1-\eta^2)}{2}$$

General formulas:

Corner nodes ($\xi \neq 0$, $\eta \neq 0$): $N_i = \frac{1}{4}(1+\xi\xi_i)(1+\eta\eta_i)(\xi\xi_i+\eta\eta_i-1)$

Nodes 5 & 7 ($\xi = 0$, $\eta = \pm 1$): $N_i = \frac{1}{2}(1-\xi^2)(1+\eta\eta_i)$

Nodes 6 & 8 ($\xi = \pm 1$, $\eta = 0$): $N_i = \frac{1}{2}(1+\xi\xi_i)(1-\eta^2)$

Example 3: Shape function of cubic serendipity square element (12-noded)
Solution

Corner node $N_1$:

Assume: $N_1 = C(1-\xi)(1-\eta)\left(\xi^2+\eta^2-\frac{10}{9}\right)$

$$1 = C(2)(2)\left(1+1-\frac{10}{9}\right) \Rightarrow C = \frac{9}{32}$$

Corner nodes:

$$N_1 = \frac{9}{32}(1-\xi)(1-\eta)\left(\xi^2+\eta^2-\frac{10}{9}\right)$$ $$N_2 = \frac{9}{32}(1+\xi)(1-\eta)\left(\xi^2+\eta^2-\frac{10}{9}\right)$$ $$N_3 = \frac{9}{32}(1+\xi)(1+\eta)\left(\xi^2+\eta^2-\frac{10}{9}\right)$$ $$N_4 = \frac{9}{32}(1-\xi)(1+\eta)\left(\xi^2+\eta^2-\frac{10}{9}\right)$$

Mid-side node $N_5$:

Assume: $N_5 = C(1-\xi)(1+\xi)(1-\eta)(1-3\xi)$

$$1 = C\left(\frac{4}{3}\right)\left(\frac{2}{3}\right)(2)(2) \Rightarrow C = \frac{9}{32}$$

Mid-side nodes:

$$N_5 = \frac{9}{32}(1-\xi^2)(1-\eta)(1-3\xi)$$ $$N_6 = \frac{9}{32}(1-\xi^2)(1-\eta)(1+3\xi)$$ $$N_7 = \frac{9}{32}(1+\xi)(1-\eta^2)(1-3\eta)$$ $$N_8 = \frac{9}{32}(1+\xi)(1-\eta^2)(1+3\eta)$$ $$N_9 = \frac{9}{32}(1-\xi^2)(1+\eta)(1+3\xi)$$ $$N_{10} = \frac{9}{32}(1-\xi^2)(1+\eta)(1-3\xi)$$ $$N_{11} = \frac{9}{32}(1-\xi)(1-\eta^2)(1+3\eta)$$ $$N_{12} = \frac{9}{32}(1-\xi)(1-\eta^2)(1-3\eta)$$

General formulas:

Corners: $N_i = \frac{9}{32}(1+\xi\xi_i)(1+\eta\eta_i)\left(\xi^2+\eta^2-\frac{10}{9}\right)$

Side nodes ($\xi \neq \pm 1$, $\eta = \pm 1$): $N_i = \frac{9}{32}(1-\xi^2)(1+\eta\eta_i)(1+9\xi\xi_i)$

Side nodes ($\xi = \pm 1$, $\eta \neq \pm 1$): $N_i = \frac{9}{32}(1+\xi\xi_i)(1-\eta^2)(1+9\eta\eta_i)$

Shape Functions for Isoparametric Triangular Elements

Shape function of three-noded triangle in terms of $\xi$, $\eta$ (Isoparametric triangle)
Solution

$N_1$ satisfies: $N_1=0$ @ $\xi+\eta-1=0$ and $N_1=1$ @ $\xi=0,\eta=0$.

Assume $N_1 = C(\xi+\eta-1)$. Apply BC: $1 = C(0+0-1) \Rightarrow C=-1$

$$N_1 = 1 - \xi - \eta, \qquad N_2 = \xi, \qquad N_3 = \eta$$
Shape function of 6-noded triangle in terms of $\xi$ & $\eta$ (Isoparametric triangle)
Solution

$N_1$ satisfies: $N_1=0$ @ $1-\xi-\eta=0$ and @ $2\xi+2\eta-1=0$; $N_1=1$ @ $\xi=0,\eta=0$.

Assume $N_1 = C(1-\xi-\eta)(2\xi+2\eta-1)$. Apply BC: $C=-1$.

$$N_1 = (1-\xi-\eta)(1-2\xi-2\eta)$$ $$N_2 = \xi(2\xi-1)$$ $$N_3 = \eta(2\eta-1)$$ $$N_4 = 4\xi(1-\xi-\eta)$$ $$N_5 = 4\xi\eta$$ $$N_6 = 4\eta(1-\xi-\eta)$$
Shape function of 6-noded triangle in terms of $L_1, L_2, L_3$
Solution

Corner nodes:

$N_1$: satisfies $N_1=0$ @ $L_1=0$ and $L_1=\frac{1}{2}$; $N_1=1$ @ $L_1=1$.

Assume $N_1 = C \cdot L_1(L_1-\frac{1}{2})$. Apply BC: $C=2$.

$$N_1 = 2L_1\left(L_1-\frac{1}{2}\right), \quad N_2 = 2L_2\left(L_2-\frac{1}{2}\right), \quad N_3 = 2L_3\left(L_3-\frac{1}{2}\right)$$

Mid-side nodes:

$N_4$: satisfies $N_4=0$ @ $L_1=0$ and $L_2=0$; $N_4=1$ @ $L_1=\frac{1}{2},L_2=\frac{1}{2}$.

Assume $N_4 = CL_1L_2$. Apply BC: $C=4$.

$$N_4 = 4L_1L_2, \qquad N_5 = 4L_2L_3, \qquad N_6 = 4L_1L_3$$
Shape function of 10-noded triangle in terms of $L_1, L_2, L_3$
Solution

Corner nodes:

$N_1$: $N_1=0$ @ $L_1=0$, $L_1-\frac{1}{3}=0$, $L_1-\frac{2}{3}=0$; $N_1=1$ @ $L_1=1$.

Assume $N_1 = CL_1(L_1-\frac{1}{3})(L_1-\frac{2}{3})$. Apply BC: $C=\frac{9}{2}$.

$$N_1 = \frac{9}{2}L_1\left(L_1-\frac{1}{3}\right)\left(L_1-\frac{2}{3}\right)$$ $$N_2 = \frac{9}{2}L_2\left(L_2-\frac{1}{3}\right)\left(L_2-\frac{2}{3}\right)$$ $$N_3 = \frac{9}{2}L_3\left(L_3-\frac{1}{3}\right)\left(L_3-\frac{2}{3}\right)$$

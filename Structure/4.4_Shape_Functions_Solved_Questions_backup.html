Q3. Derive the shape function for the 10-noded triangular element  [6] — 2078 Chaitra, Q4, Group B
Derive the shape functions for the element as shown in figure (a 10-noded triangular element with 3 corner nodes, 6 side nodes, and 1 interior node).
10-noded cubic triangular element with area coordinates $(L_1, L_2, L_3)$
Solution

We use area (natural) coordinates $L_1, L_2, L_3$ with the constraint $L_1 + L_2 + L_3 = 1$. The element is a complete cubic triangle requiring $m = \frac{(3+1)(3+2)}{2} = 10$ terms.

Corner Nodes ($L_i = 1$ at vertex $i$, cubic in $L_i$):

For $N_1$: must be zero at $L_1 = 0$, $L_1 = \frac{1}{3}$, and $L_1 = \frac{2}{3}$, and equal to 1 at $L_1 = 1$.

$$N_1 = C \cdot L_1 \left(L_1 - \frac{1}{3}\right)\left(L_1 - \frac{2}{3}\right)$$

At $L_1 = 1$: $1 = C(1)\left(\frac{2}{3}\right)\left(\frac{1}{3}\right) = \frac{2C}{9} \implies C = \frac{9}{2}$

$$N_1 = \frac{9}{2}L_1\left(L_1 - \frac{1}{3}\right)\left(L_1 - \frac{2}{3}\right)$$ $$N_2 = \frac{9}{2}L_2\left(L_2 - \frac{1}{3}\right)\left(L_2 - \frac{2}{3}\right)$$ $$N_3 = \frac{9}{2}L_3\left(L_3 - \frac{1}{3}\right)\left(L_3 - \frac{2}{3}\right)$$

Side Nodes (between corners, at $\frac{1}{3}$ and $\frac{2}{3}$ positions):

For $N_4$ (between nodes 1 and 2, at $L_1 = \frac{2}{3}$, $L_2 = \frac{1}{3}$):

$$N_4 = C \cdot L_1 L_2 \left(L_1 - \frac{1}{3}\right)$$

At node 4: $1 = C\left(\frac{2}{3}\right)\left(\frac{1}{3}\right)\left(\frac{1}{3}\right) \implies C = \frac{27}{2}$

$$N_4 = \frac{27}{2}L_1 L_2\left(L_1 - \frac{1}{3}\right), \qquad N_5 = \frac{27}{2}L_1 L_2\left(L_2 - \frac{1}{3}\right)$$ $$N_6 = \frac{27}{2}L_2 L_3\left(L_2 - \frac{1}{3}\right), \qquad N_7 = \frac{27}{2}L_2 L_3\left(L_3 - \frac{1}{3}\right)$$ $$N_8 = \frac{27}{2}L_1 L_3\left(L_3 - \frac{1}{3}\right), \qquad N_9 = \frac{27}{2}L_1 L_3\left(L_1 - \frac{1}{3}\right)$$

Interior Node (centroid at $L_1 = L_2 = L_3 = \frac{1}{3}$):

$$N_{10} = C \cdot L_1 L_2 L_3$$

At node 10: $1 = C\left(\frac{1}{3}\right)^3 = \frac{C}{27} \implies C = 27$

$$\boxed{N_{10} = 27\,L_1 L_2 L_3}$$

This completes the 10 shape functions for the cubic triangular element.

Q4. Derive the shape functions for the given eight-noded rectangular element  [6] — 2078 Kartik, Q4, Group B
Derive the shape functions for the 8-noded rectangular element (Serendipity element) with nodes at corners and mid-sides in natural coordinates $\xi, \eta \in [-1, +1]$.
8-noded serendipity rectangular element
Solution

We use the Serendipity concept (Ref: Section 4.4 — Serendipity Family). Each $N_i$ must equal 1 at node $i$ and 0 at all other nodes.

Corner Nodes:

For $N_1$ at $(\xi_1, \eta_1) = (-1, -1)$:

  • $N_1 = 0$ when $\xi = 1 \implies (1-\xi)$ is a factor
  • $N_1 = 0$ when $\eta = 1 \implies (1-\eta)$ is a factor
  • $N_1 = 0$ at node 5 ($\xi=0,\eta=-1$) and node 8 ($\xi=-1,\eta=0$): the line through these mid-side nodes is $\xi + \eta + 1 = 0$, so $(1+\xi+\eta)$ is a factor

Assume $N_1 = C(1-\xi)(1-\eta)(1+\xi+\eta)$.

At node 1: $1 = C(1+1)(1+1)(1-1-1) = C(2)(2)(-1) = -4C \implies C = -\frac{1}{4}$

$$N_1 = -\frac{1}{4}(1-\xi)(1-\eta)(1+\xi+\eta)$$

By similar reasoning for the other corners:

$$N_1 = -\frac{1}{4}(1-\xi)(1-\eta)(1+\xi+\eta)$$ $$N_2 = -\frac{1}{4}(1+\xi)(1-\eta)(1-\xi+\eta)$$ $$N_3 = -\frac{1}{4}(1+\xi)(1+\eta)(1-\xi-\eta)$$ $$N_4 = -\frac{1}{4}(1-\xi)(1+\eta)(1+\xi-\eta)$$

General corner formula: $N_i = \frac{1}{4}(1+\xi\xi_i)(1+\eta\eta_i)(\xi\xi_i + \eta\eta_i - 1)$

Mid-side Nodes:

For $N_5$ at $(0, -1)$:

  • $N_5 = 0$ when $\xi = \pm 1 \implies (1-\xi^2)$ is a factor
  • $N_5 = 0$ when $\eta = 1 \implies (1-\eta)$ is a factor

Assume $N_5 = C(1-\xi^2)(1-\eta)$. At node 5: $1 = C(1)(2) \implies C = \frac{1}{2}$

$$N_5 = \frac{1}{2}(1-\xi^2)(1-\eta)$$ $$N_6 = \frac{1}{2}(1+\xi)(1-\eta^2)$$ $$N_7 = \frac{1}{2}(1-\xi^2)(1+\eta)$$ $$N_8 = \frac{1}{2}(1-\xi)(1-\eta^2)$$

Summary of general formulas for 8-noded serendipity element:

$$\text{Corners } (\xi_i = \pm 1, \eta_i = \pm 1): \quad N_i = \frac{1}{4}(1+\xi\xi_i)(1+\eta\eta_i)(\xi\xi_i + \eta\eta_i - 1)$$ $$\text{Mid-sides } (\xi_i = 0, \eta_i = \pm 1): \quad N_i = \frac{1}{2}(1-\xi^2)(1+\eta\eta_i)$$ $$\text{Mid-sides } (\xi_i = \pm 1, \eta_i = 0): \quad N_i = \frac{1}{2}(1+\xi\xi_i)(1-\eta^2)$$
Q5. Obtain the shape function for the triangular element shown  [6] — 2076 Bhadra, Q9, Group B
Obtain the shape functions for a 3-noded triangular element (CST — Constant Strain Triangle).
3-noded CST element
Solution

Method 1: Polynomial (Cartesian Coordinate) Approach

6 nodal displacements ($u_1, u_2, u_3, v_1, v_2, v_3$) $\implies$ select polynomial with 3 coefficients each:

$$u = \alpha_1 + \alpha_2 x + \alpha_3 y, \qquad v = \alpha_4 + \alpha_5 x + \alpha_6 y$$

Applying boundary conditions for $u$:

$$\begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix} = \begin{bmatrix} 1 & x_1 & y_1 \\ 1 & x_2 & y_2 \\ 1 & x_3 & y_3 \end{bmatrix} \begin{Bmatrix} \alpha_1 \\ \alpha_2 \\ \alpha_3 \end{Bmatrix}$$

Inverting:

$$\begin{Bmatrix} \alpha_1 \\ \alpha_2 \\ \alpha_3 \end{Bmatrix} = \frac{1}{2A} \begin{bmatrix} a_1 & a_2 & a_3 \\ b_1 & b_2 & b_3 \\ c_1 & c_2 & c_3 \end{bmatrix} \begin{Bmatrix} u_1 \\ u_2 \\ u_3 \end{Bmatrix}$$

where $2A = \det \begin{vmatrix} 1 & x_1 & y_1 \\ 1 & x_2 & y_2 \\ 1 & x_3 & y_3 \end{vmatrix}$ and:

Constants (cyclic permutation of 1, 2, 3)

$a_i = x_j y_k - x_k y_j$,   $b_i = y_j - y_k$,   $c_i = x_k - x_j$

where $(i, j, k)$ = cyclic: $(1,2,3)$, $(2,3,1)$, $(3,1,2)$

Substituting back: $u = [1 \;\; x \;\; y] \cdot \frac{1}{2A}[C]\{u_i\}$

$$\boxed{N_i = \frac{1}{2A}(a_i + b_i x + c_i y)} \quad \text{for } i = 1, 2, 3$$

where $a_1 = x_2 y_3 - x_3 y_2$, $b_1 = y_2 - y_3$, $c_1 = x_3 - x_2$, etc.

Method 2: Natural (Area) Coordinate Approach

Using area coordinates $L_1, L_2, L_3$ with $L_1 + L_2 + L_3 = 1$:

$$u = \alpha_1 L_1 + \alpha_2 L_2 + \alpha_3 L_3$$

At node 1 $(L_1=1, L_2=0, L_3=0)$: $u_1 = \alpha_1$. Similarly $u_2 = \alpha_2$, $u_3 = \alpha_3$.

$$\boxed{N_1 = L_1, \quad N_2 = L_2, \quad N_3 = L_3}$$
Q6. Derive the shape functions for the eight-noded 2-D rectangular element  [8] — 2075 Bhadra, Q10, Group B
Derive shape functions for the eight-noded 2D rectangular element with dimensions $2a \times 2b$ centered at the origin, with nodes at corners and mid-sides.
8-noded rectangular element with dimensions $2a \times 2b$
Solution

This is identical to Q4 above. Using the natural coordinate mapping $\xi = x/a$, $\eta = y/b$ (so that $\xi, \eta \in [-1,1]$), the derivation follows the Serendipity approach from Section 4.4.

The shape functions are derived exactly as in Q4. For completeness, restating the results:

Corner Nodes ($\xi_i = \pm 1$, $\eta_i = \pm 1$):

$$N_i = \frac{1}{4}(1+\xi\xi_i)(1+\eta\eta_i)(\xi\xi_i + \eta\eta_i - 1)$$

Mid-side Nodes:

$$\text{Nodes 5, 7 } (\xi_i = 0): \quad N_i = \frac{1}{2}(1-\xi^2)(1+\eta\eta_i)$$ $$\text{Nodes 6, 8 } (\eta_i = 0): \quad N_i = \frac{1}{2}(1+\xi\xi_i)(1-\eta^2)$$

Verification:

Node$(\xi_i, \eta_i)$$N_i$ at that node
1$(-1,-1)$$\frac{1}{4}(2)(2)(-1-1-1) = -\frac{1}{4}(4)(-1) = 1$ ✓
5$(0,-1)$$\frac{1}{2}(1-0)(1+1) = 1$ ✓

Also verify partition of unity: $\sum_{i=1}^{8} N_i = 1$ for all $(\xi, \eta)$. ✓

Q7. Obtain shape functions $N_i$ for the eight-noded rectangular element  [5] — 2074 Bhadra, Q9, Group B
Obtain shape functions $N_i$ for the eight-noded rectangular element as shown (with natural coordinates $\xi, \eta$).
Solution

This is again the 8-noded serendipity element as derived in Q4 and Q6. The Serendipity approach yields:

Corner nodes ($i = 1, 2, 3, 4$):

$$N_i = \frac{1}{4}(1+\xi_0)(1+\eta_0)(\xi_0 + \eta_0 - 1)$$

where $\xi_0 = \xi\xi_i$, $\eta_0 = \eta\eta_i$.

Expanding individually:

$$N_1 = -\frac{1}{4}(1-\xi)(1-\eta)(1+\xi+\eta) \quad \text{at } (-1,-1)$$ $$N_2 = -\frac{1}{4}(1+\xi)(1-\eta)(1-\xi+\eta) \quad \text{at } (1,-1)$$ $$N_3 = -\frac{1}{4}(1+\xi)(1+\eta)(1-\xi-\eta) \quad \text{at } (1,1)$$ $$N_4 = -\frac{1}{4}(1-\xi)(1+\eta)(1+\xi-\eta) \quad \text{at } (-1,1)$$

Mid-side nodes ($i = 5, 6, 7, 8$):

$$N_5 = \frac{1}{2}(1-\xi^2)(1-\eta) \quad \text{at } (0,-1)$$ $$N_6 = \frac{1}{2}(1+\xi)(1-\eta^2) \quad \text{at } (1,0)$$ $$N_7 = \frac{1}{2}(1-\xi^2)(1+\eta) \quad \text{at } (0,1)$$ $$N_8 = \frac{1}{2}(1-\xi)(1-\eta^2) \quad \text{at } (-1,0)$$

Property check: $N_i(\xi_j, \eta_j) = \delta_{ij}$ (Kronecker delta). Each $N_i = 1$ at node $i$ and $0$ at all other nodes. ✓

Q8. Derive shape functions of a three-noded iso-parametric triangular element  [2] — 2074 Magh, Q8, Group B
Derive the shape functions of a three-noded isoparametric triangular element.
3-noded isoparametric triangle in $(\xi, \eta)$ space
Solution

The parent element has vertices at $(0,0)$, $(1,0)$, $(0,1)$ with constraint $\xi + \eta \leq 1$.

For $N_1$ at $(0,0)$: Must be zero on the hypotenuse $\xi + \eta - 1 = 0$.

$$N_1 = C(\xi + \eta - 1), \quad \text{At node 1: } 1 = C(0+0-1) \implies C = -1$$

For $N_2$ at $(1,0)$: Must be zero at $\xi = 0$.

$$N_2 = C \cdot \xi, \quad \text{At node 2: } 1 = C(1) \implies C = 1$$

For $N_3$ at $(0,1)$: Must be zero at $\eta = 0$.

$$N_3 = C \cdot \eta, \quad \text{At node 3: } 1 = C(1) \implies C = 1$$
$$\boxed{N_1 = 1 - \xi - \eta, \qquad N_2 = \xi, \qquad N_3 = \eta}$$

Verification: $N_1 + N_2 + N_3 = (1 - \xi - \eta) + \xi + \eta = 1$ ✓

Q9. Derive the shape functions $N_i$ for the six-noded triangular 2D element  [6] — 2073 Bhadra, Q9, Group B
Derive the shape functions $N_i$ for the six-noded triangular 2D element shown in the figure, with 3 corner nodes and 3 mid-side nodes. Base = $2a$, height = $2b$, origin at mid-point of base.
6-noded quadratic triangular element with Cartesian coordinates
Solution

Step 1: Node Coordinates

From the figure, with origin at the midpoint of the base:

Node$x$$y$Type
1$-a$$0$Corner
2$a$$0$Corner
3$0$$2b$Corner
4$0$$0$Mid-side (1–2)
5$a/2$$b$Mid-side (2–3)
6$-a/2$$b$Mid-side (3–1)

Step 2: Area Coordinates for this Triangle

Vertices: $(x_1, y_1) = (-a, 0)$, $(x_2, y_2) = (a, 0)$, $(x_3, y_3) = (0, 2b)$.

$$2A = \begin{vmatrix} 1 & -a & 0 \\ 1 & a & 0 \\ 1 & 0 & 2b \end{vmatrix} = 1(2ab - 0) - (-a)(2b - 0) + 0 = 2ab + 2ab = 4ab$$

So the total area $A = 2ab$.

The area coordinates $L_i = \frac{1}{2A}(a_i + b_i x + c_i y)$ where:

Computing $a_i, b_i, c_i$ with cyclic $(i,j,k)$:
$a_i = x_j y_k - x_k y_j$$b_i = y_j - y_k$$c_i = x_k - x_j$
$i=1,\;j=2,\;k=3$$a(2b) - 0 = 2ab$$0 - 2b = -2b$$0 - a = -a$
$i=2,\;j=3,\;k=1$$0 - (-a)(2b) = 2ab$$2b - 0 = 2b$$-a - 0 = -a$
$i=3,\;j=1,\;k=2$$(-a)(0) - a(0) = 0$$0 - 0 = 0$$a - (-a) = 2a$
$$L_1 = \frac{1}{4ab}(2ab - 2bx - ay) = \frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}$$ $$L_2 = \frac{1}{4ab}(2ab + 2bx - ay) = \frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}$$ $$L_3 = \frac{1}{4ab}(0 + 0 + 2ay) = \frac{y}{2b}$$
Verification of $L_i$:
• At node 1 $(-a,0)$: $L_1 = \frac{1}{2}+\frac{1}{2}-0 = 1$, $L_2 = \frac{1}{2}-\frac{1}{2}-0 = 0$, $L_3 = 0$ ✓
• At node 2 $(a,0)$: $L_1 = 0$, $L_2 = 1$, $L_3 = 0$ ✓
• At node 3 $(0,2b)$: $L_1 = \frac{1}{2}-0-\frac{1}{2} = 0$, $L_2 = 0$, $L_3 = 1$ ✓
• $L_1 + L_2 + L_3 = 1 - \frac{y}{2b} + \frac{y}{2b} = 1$ ✓

Step 3: Shape Functions in Area Coordinates

For a 6-noded quadratic triangle, using the standard area coordinate formulas (Ref: Sec. 4.4):

Corner nodes ($N_i$ must vanish at $L_i = 0$ and $L_i = \frac{1}{2}$):

$$N_i = L_i(2L_i - 1) \quad \text{for } i = 1, 2, 3$$

Mid-side nodes ($N_k$ at midpoint of side between $i$ and $j$):

$$N_4 = 4L_1 L_2, \quad N_5 = 4L_2 L_3, \quad N_6 = 4L_1 L_3$$

Step 4: Shape Functions in Cartesian Coordinates $(x, y)$

Substituting the expressions for $L_1, L_2, L_3$:

$$N_1 = L_1(2L_1 - 1) = \left(\frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}\right)\!\left(- \frac{x}{a} - \frac{y}{2b}\right)$$ $$N_2 = L_2(2L_2 - 1) = \left(\frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}\right)\!\left(\frac{x}{a} - \frac{y}{2b}\right)$$ $$N_3 = L_3(2L_3 - 1) = \frac{y}{2b}\!\left(\frac{y}{b} - 1\right)$$
$$N_4 = 4L_1 L_2 = 4\left(\frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}\right)\!\left(\frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}\right)$$ $$= \left(1 - \frac{y}{2b}\right)^2 - \frac{x^2}{a^2}$$ $$N_5 = 4L_2 L_3 = 4\left(\frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}\right)\frac{y}{2b} = \frac{2y}{b}\left(\frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}\right)$$ $$N_6 = 4L_1 L_3 = 4\left(\frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}\right)\frac{y}{2b} = \frac{2y}{b}\left(\frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}\right)$$

Final shape functions (area coordinate form):

$$\boxed{N_1 = L_1(2L_1-1), \quad N_2 = L_2(2L_2-1), \quad N_3 = L_3(2L_3-1)}$$ $$\boxed{N_4 = 4L_1 L_2, \quad N_5 = 4L_2 L_3, \quad N_6 = 4L_1 L_3}$$

where $\;L_1 = \frac{1}{2} - \frac{x}{2a} - \frac{y}{4b}$, $\;L_2 = \frac{1}{2} + \frac{x}{2a} - \frac{y}{4b}$, $\;L_3 = \frac{y}{2b}$

Verification at nodes:

Node$(x, y)$$(L_1, L_2, L_3)$$N_i = 1$?
1$(-a, 0)$$(1, 0, 0)$$N_1 = 1(2-1)=1$ ✓
2$(a, 0)$$(0, 1, 0)$$N_2 = 1(2-1)=1$ ✓
3$(0, 2b)$$(0, 0, 1)$$N_3 = 1(2-1)=1$ ✓
4$(0, 0)$$(\frac{1}{2}, \frac{1}{2}, 0)$$N_4 = 4 \cdot \frac{1}{2} \cdot \frac{1}{2} = 1$ ✓
5$(a/2, b)$$(0, \frac{1}{2}, \frac{1}{2})$$N_5 = 4 \cdot \frac{1}{2} \cdot \frac{1}{2} = 1$ ✓
6$(-a/2, b)$$(\frac{1}{2}, 0, \frac{1}{2})$$N_6 = 4 \cdot \frac{1}{2} \cdot \frac{1}{2} = 1$ ✓
Q10. Derive the shape function for the element shown  [8] — 2072 Ashwin, Q7b, Group B
Derive the shape functions for a 6-noded triangular element using area (natural) coordinates $L_1, L_2, L_3$. The figure shows a 6-noded triangle with values of $L_i$ marked at each node.
6-noded triangle with area coordinates $L_1, L_2, L_3$
Solution

Using area coordinates $L_1, L_2, L_3$ with $L_1 + L_2 + L_3 = 1$ (Ref: Section 4.4 — Shape Functions for Isoparametric Triangular Elements).

Corner Nodes:

$N_1$ at node 1 ($L_1=1, L_2=0, L_3=0$): Must vanish when $L_1 = 0$ and when $L_1 = \frac{1}{2}$.

$$N_1 = C \cdot L_1\left(L_1 - \frac{1}{2}\right)$$ $$\text{At node 1 } (L_1=1): \quad 1 = C \cdot 1 \cdot \frac{1}{2} \implies C = 2$$
$$N_1 = 2L_1\left(L_1 - \frac{1}{2}\right) = L_1(2L_1 - 1)$$ $$N_2 = 2L_2\left(L_2 - \frac{1}{2}\right) = L_2(2L_2 - 1)$$ $$N_3 = 2L_3\left(L_3 - \frac{1}{2}\right) = L_3(2L_3 - 1)$$

Mid-side Nodes:

$N_4$ at node 4 ($L_1 = \frac{1}{2}, L_2 = \frac{1}{2}, L_3 = 0$): Must vanish when $L_1 = 0$ and $L_2 = 0$.

$$N_4 = C \cdot L_1 L_2$$ $$\text{At node 4: } 1 = C \cdot \frac{1}{2} \cdot \frac{1}{2} \implies C = 4$$

Similarly for nodes 5 and 6:

Corner nodes:

$$\boxed{N_1 = L_1(2L_1 - 1), \quad N_2 = L_2(2L_2 - 1), \quad N_3 = L_3(2L_3 - 1)}$$

Mid-side nodes:

$$\boxed{N_4 = 4L_1 L_2, \quad N_5 = 4L_2 L_3, \quad N_6 = 4L_1 L_3}$$

Verification:

Node$(L_1, L_2, L_3)$$N_i$ value
1$(1, 0, 0)$$N_1 = 1(2-1) = 1$, all others = 0 ✓
4$(\frac{1}{2}, \frac{1}{2}, 0)$$N_4 = 4 \cdot \frac{1}{2} \cdot \frac{1}{2} = 1$, $N_1 = \frac{1}{2}(0) = 0$ ✓
5$(0, \frac{1}{2}, \frac{1}{2})$$N_5 = 4 \cdot \frac{1}{2} \cdot \frac{1}{2} = 1$, $N_2 = \frac{1}{2}(0) = 0$ ✓
Partition of unity: $\sum_{i=1}^{6} N_i = 1$ for all $(L_1, L_2, L_3)$. This can be verified by algebraic expansion using $L_1 + L_2 + L_3 = 1$.
